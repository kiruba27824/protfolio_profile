
# Personal Portfolio

## Overview

This is my personal portfolio website developed to showcase my skills, projects, certifications, and educational background as an aspiring Python Full Stack Developer. The portfolio highlights my technical expertise and serves as a professional platform for recruiters and employers to learn more about my work.

## Features

- Responsive and modern user interface
- Professional About Me section
- Technical Skills showcase
- Project portfolio with descriptions
- Education and Certifications
- Resume download option
- Contact section
- Smooth scrolling and animations
- Mobile-friendly design

## Technologies Used

- HTML5
- CSS3
- JavaScript
- Bootstrap
- Font Awesome
- Google Fonts

## Installation

1. Clone the repository.

```
git clone https://github.com/your-username/portfolio.git
```

2. Navigate to the project folder.

```
cd portfolio
```

3. Open `index.html` in your web browser.
